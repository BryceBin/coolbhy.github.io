C++ STL 总结

STL(Standard Template Library):标准模版库
cpp文档：https://zh.cppreference.com/w/cpp

容器

vector 主要函数

增加：

  • void push_back((const T& x):向量尾部添加元素X
  • iterator insert(iterator it,const T& x):向量中迭代器指向元素前增加一个元素x

    删除：

  • iterator erase(iterator it):删除向量中迭代器指向元素
  • iterator erase(iterator first,iterator last):删除向量中[first,last)中元素
  • void pop_back():删除向量中最后一个元素
  • void clear():清空向量中所有元素

    访问：

  • reference at(int pos):返回pos位置元素的引用
  • reference front():返回首元素的引用
  • reference back():返回尾元素的引用
  • for(vector<int>::iterator it=vec.begin();it!=vec.end();it++):遍历

    大小：

  • int size() const:返回向量中元素的个数
  • int capacity() const:返回当前向量张红所能容纳的最大元素值
  • int max_size() const:返回最大可允许的vector元素数量值

Map(key->value):键值关联型容器

构造

  • map<Type,Type> m;

    插入

  • m.insert(pair<Type,Type>(key,value));
  • m.insert(map<Type,Type>::value_type(key,value));
  • m[key]=value;

    大小

  • size():返回map中的元素个数

    删除

  • clear():清空map
  • erase(iterator pos):删除位于pos处的元素

    查找

  • count(const Key &key):map中有该键则返回1，否则返回0
  • find(const Key &key):返回该键的迭代器，无则返回m.end();

    multimap:允许多个入口拥有同一关键

pair(对组)

类模板：template struct pair
功能：pair将一对值（可以是不同的数据类型）组合成一个值，两个值可以分 别用pair的两个公有函数first和second访问。

  • 初始化

  • 访问

  • 赋值

set与multiset

set是一个有序集合，默认以升序排序，支持插入、删除和查找等操作。set与multiset的区别就是set插入的元素不能重复，而multiset可以。

大小、判空：

  • int size() const:返回容器元素个数
  • bool empty() const:判断容器是否为空，若返回true，表明容器已空

    插入、删除：

  • pair insert( x):插入元素x
  • iterator insert(iterator it,x):在迭代器it处插入元素x
  • void insert(const value_type first,const value_type last):插入[first, last)之间元素
  • size_type erase(const Key& key):删除元素值等于key的元素
  • iterator erase(iterator it):删除迭代器指针it处元素
  • iterator erase(iterator first,iterator last):删除[first, last)之间元素

    定义比较函数

priority_queue:优先队列

与队尾进队头出的普通队列相比，优先队列还能够自动排序。当插入一个元素时，优先队列将对其根据设置的比较函数进行排序，默认情况下是升序排序，即队头为最小元素，队尾为最大元素。

算法

<algorithm>库的常用函数

  • InputIterator find (InputIterator first, InputIterator last, const T& val)：找到val就返回该元素的迭代器，否则返回last;

  • sort(first,end,method):method可选
    其中first和end可以是地址也可以是迭代器

  • ForwardIterator lower_bound (ForwardIterator first, ForwardIterator last,const T& val);
    对于给定的一个已经排好序的序列，返回该范围内第一个不小于val的值的迭代器，即下界

  • ForwardIterator upper_bound (ForwardIterator first, ForwardIterator last,const T& val);
    对于给定的一个已经排好序的序列，返回该范围内第一个大于val的值的迭代器，即上界
  • next_permutation(vec.begin(),vec.end())
    对给定的序列进行全排列，将序列按照字典序排列得到下一个排列，若当前序列为最后一个排列，则返回false;
    同理还有pre_permutation
